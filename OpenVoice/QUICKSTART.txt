======================================
  OpenVoice AAC Communicator
  Quick-Start Guide
======================================

HOW TO START
------------
Double-click  OpenVoice.bat  in this folder.

That's it. No installation. No internet. Just double-click and go.


WHAT YOU'LL SEE
---------------
A menu with 6 options:

  1. Type Message     -- Type anything and hear it spoken
  2. Quick Phrases    -- Pick from the 12-phrase board (arrow keys + Enter)
  3. Voice Settings   -- Change pitch, speed, and volume
  4. Export WAV       -- Save speech as an audio file
  5. About            -- App info
  6. Exit             -- Close the app


KEYS
----
  UP / DOWN arrows  = move through the menu
  LEFT / RIGHT      = move between phrases or change a slider value
  ENTER             = confirm / speak
  1 through 6       = jump straight to a menu item
  ESC               = go back / cancel
  S                 = save voice settings
  Q                 = quit


VOICE SETTINGS
--------------
Pitch:   -10 to +10 semitones  (negative = lower, positive = higher)
Speed:   80 to 450 wpm         (175 = normal conversational pace)
Volume:  0 to 200              (100 = your system's normal level)

Settings are saved automatically when you press S or when you exit.


CUSTOM PHRASES
--------------
Open the file  data\phrases.txt  in Notepad.
Add one phrase per line.
Lines starting with # are ignored.
Save the file, then restart the app.


WAV FILES
---------
Use Export WAV to save speech as a .wav audio file.
Files are saved in the OpenVoice folder with a date-and-time name, e.g.:
  openvoice_20260614_160300.wav


REQUIREMENTS
------------
- Windows 10 or Windows 11
- Speakers or headphones plugged in
- Nothing else needs to be installed


TROUBLESHOOTING
---------------
No sound?
  -> Check your Windows volume is turned up and not muted.
  -> Make sure speakers/headphones are connected.

Phrases not showing?
  -> Make sure you double-clicked OpenVoice.bat from inside
     the OpenVoice folder, not a copy of the .exe moved elsewhere.

Profile not saving?
  -> The data\ folder must be next to openvoice.exe.
